## 2. 서술 기법

### 초현실적 요소의 자연스러운 통합
- 비현실적 사건을 일상적인 것처럼 묘사
- 시간과 공간의 왜곡을 등장인물이 당연하게 받아들이도록 설정
- 꿈과 현실의 경계가 모호한 장면 포함
- 반복되는 패턴과 순환적 구조 활용

### 암시적 표현
- 직접적인 언급 대신 은유와 상징 사용
- 꿈을 암시하는 모티프 활용 (거울, 물, 반복, 변형 등)
- 기억의 불확실성과 주관성 강조
- 다층적 해석이 가능한 대사와 상황 설정

## 3. 금기 사항

### 절대 해서는 안 되는 것(금지사항)
- 전체가 꿈이라는 직접적 언급 금지
- 깨어날 수 있다는 명시적 암시 금지
- 꿈을 꾸는 주체의 정체 밝히는것 금지
- 현실 세계와의 직접적 연결 설정 금지
### 주의해야 할 것
- 지나치게 노골적인 초현실적 요소
- 등장인물의 완전한 각성
- 논리적 모순의 과도한 노출
- 꿈의 구조를 설명하려는 시도
