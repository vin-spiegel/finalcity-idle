# 파이널 시티: 자동 탐색 로그라이크

## 핵심 컨셉
- 마나로 오염된 버려진 도시를 탐험하는 1인 로그라이크
- 현대적 배경과 장비를 활용한 생존 게임
- 자동 전투/탐색 시스템으로 전략적 의사결정에 집중
- 돌발적인 특성 변화와 상황 발생으로 매 런이 다른 경험 제공

## 게임플레이 시스템

### 캐릭터 특성
시작 시 무작위로 2개의 특성 부여, 레벨업 시 1개 추가 획득
- 총 덕후: 사격 지연 -10%, 총기 장착 시 시야 +10
- 방사능 저항: 마나 피폭 저항 +50%, 돌연변이화 속도 -30%
- 구울: 마나 피폭 면역, 시체 섭취로 회복 가능, 사회성 -20%
- 식탐: 허기 +150%, 음식 효과 2배
- 빠른 손: 재장전 속도 +15%, 근접 정확도 +5%
- 천재: 스킬 습득 속도 2배, 작업 속도 -25%
- 매력적: NPC 대화 성공률 +10%, 상점 가격 -5%
- 튼튼한 허리: 소지품 무게 한도 +50%

### 자동 행동 시스템
1. 전투 모드
- 자동 조준 및 발사
- 탄약 관리 및 재장전
- HP에 따른 자동 회피/후퇴
- 무기 상황별 교체

2. 탐색 모드
- 지정된 방향으로 자동 이동
- 함정/비밀문 자동 탐지
- 아이템 자동 수집
- 위험 감지 시 정지

### 상태 관리
1. 체력(HP)
- 현대식 의료품으로 회복
- 부상 시스템: 골절, 출혈, 감염 등
- 자동 치료 아이템 사용

2. 마나 피폭
- 시간 경과와 특정 구역에서 상승
- 100% 도달 시 무작위 변이 발생
- 정화된 물로 수치 감소

3. 영혼
- 사망 시 소모하여 부활
- 특수 능력 사용 시 소모
- 완전 소진 시 영구 사망

### 돌발 상황
1. 변이 이벤트
- 피폭으로 인한 무작위 신체 변화
- 새로운 특성 획득/기존 특성 변형
- 외형/능력 변화

2. 환경 이벤트
- 마나 폭풍: 피폭 급증, 시야 제한
- 변이체 습격: 다수의 적 출현
- 차원 균열: 이상 공간 출현

3. NPC 이벤트
- 생존자 조우: 거래/전투 선택
- 변이체 대화: 특수 퀘스트 발생
- 적대적 NPC: 함정/매복

### 장비 시스템
1. 무기
- 근접: 야구 배트, 소방도끼, 군용 나이프
- 원거리: 권총, 산탄총, 돌격소총
- 특수: 화염병, 수류탄, 지뢰

2. 방어구
- 일반: 방탄조끼, 전술조끼, 헬멧
- 특수: 방사능 슈트, 마나 차단복

3. 도구
- 의료: 구급상자, 붕대, 진통제
- 생존: 정화된 물, 랜턴, 해독제
- 탐색: 마나 감지기, 지도, 쌍안경

### 도시 구조와 난이도

1. 구획 구성
- 게임 시작 시 모든 구획 자유롭게 탐험 가능
- 진행도에 따라 모든 구획의 난이도가 점진적으로 상승
- 봉인석 3개는 무작위 구획의 특별 장소에 배치

2. 진행 기반 난이도 시스템
- 첫 방문 구획: 기본 난이도 (약한 변이체, 기본 아이템)
- 두 번째 구획: 난이도 +1 (일반 변이체, 중급 아이템)
- 세 번째 구획: 난이도 +2 (강한 변이체, 고급 아이템)
- 이후 구획: 난이도 +3 (엘리트 변이체, 희귀 아이템)

3. 난이도 영향 요소
- 변이체 능력치와 종류
- 마나 오염 상승 속도
- 아이템 품질과 희귀도
- 함정과 위험요소 빈도

4. 구획 종류
- 주거 구획
  * 아파트, 주택가, 지하 대피소
  * 생존 물자와 의료품
  * 좁은 실내 구조
- 상업 구획
  * 상점, 사무실, 은행, 병원
  * 장비와 특수 아이템
  * 복잡한 실내 구조
- 공업 구획
  * 공장, 창고, 발전소
  * 군수품과 연구 자료
  * 넓은 개방 구조
- 항구 구획
  * 부두, 컨테이너 야드
  * 희귀 장비와 물자
  * 반개방 구조
- 도심 구획
  * 고층 빌딩, 지하철, 공원
  * 다양한 아이템
  * 수직 구조
- 특수 구획
  * 군사 기지, 연구소
  * 고급 장비
  * 요새형 구조

5. 진행 흐름
- 첫 구획은 안전하게 탐험하며 기본 장비 획득
- 이후 구획은 난이도가 높아지지만 더 좋은 보상 제공
- 봉인석 3개 수집이 주요 목표
- 구획 간 자유로운 이동으로 전략적 선택 가능
- 마나원 구역은 최고 난이도로 고정

6. 정보 제공
- 현재까지의 진행 단계 표시
- 다음 구획의 예상 난이도 정보
- 구획별 주요 특징과 구조 정보
- 봉인석 보유 현황

4. 마나원 구역 개방 조건
- 봉인석 3개 수집 시 자동으로 도시 중앙에 출현
- 개방 시 모든 구획에 특수 경고 발생
- 마나원으로 가는 경로 자동 생성
- 진입 후에는 퇴각 불가능

5. 구획 정보 제공
- 맵에서 각 구획의 현재 위험도 표시
- 구획별 주요 아이템과 특징 정보 확인 가능
- 캐릭터 상태 기반 추천 구획 표시
- 봉인석 보유 현황과 남은 위치 힌트

### 엔딩 시스템

1. 메인 엔딩: 도시 심층부 마나원 봉인
- 도시의 각 구역에서 3개의 마나 봉인석 수집
  * 주거구역: 지하 대피소
  * 상업구역: 은행 금고
  * 항구지역: 컨테이너 터미널
- 봉인석을 모두 모아 도시 심층부로 진입
- 심층부에서 마나원 봉인 의식 수행
- 성공 시 도시 정화 달성

2. 게임 오버 조건
- 영혼 완전 소진으로 인한 영구 사망
- 마나 피폭 100% 도달로 인한 이성 상실
- 체력 0 도달 시 영혼 1개 소모 후 부활

### 진행 시스템
1. 목표
- 엔딩 조건 달성을 위한 아이템/상태 수집
- 구역별 중간 목표 달성
- 생존자 구출로 추가 정보/아이템 획득

2. 성장
- 레벨업: 특성 추가 획득
- 장비 개선: 제작/강화
- 기술 습득: 전투/생존 스킬

3. 점수
- 생존 시간
- 처치한 적
- 수집한 아이템
- 달성한 목표

### 시드 시스템

1. 도시 시드 정보
- 게임 시작 시 8자리 시드 번호 표시 (예: #A7B2C3D4)
- 시드 번호로 동일한 도시 재생성 가능
- 시드 정보는 언제든 메뉴에서 확인 가능

2. 시드가 결정하는 요소
- 도시 구획 배치와 구조
- 봉인석 위치
- 특수 아이템 출현
- 변이체 스폰 패턴
- 시작 시 무작위 특성 2개

3. 시드 활용
- 특정 시드 직접 입력 가능
- 흥미로운 구성 공유 가능
- 도전 과제용 고정 시드 제공
- 일일 도전용 통일 시드

4. 시드 기록
- 성공한 시드 자동 저장
- 플레이 시간과 결과 기록
- 특별한 이벤트 발생 시 시드 기록 권장

2. 이벤트 선택
- NPC 대화 선택지
- 특수 이벤트 결정
- 거래/전투 선택
- 퀘스트 수락 여부

3. 전략적 판단
- 탐사 방향 결정 (화살표로 지정)
- 주요 목표 설정
- 긴급 탈출 명령
- 아이템 사용 강제 지시

### AI 자동화 시스템

1. 전투 AI
- 자동 조준 및 발사
- 커버 활용
- 탄약 관리 및 재장전
- 적 우선순위 판단
- 근접/원거리 전투 전환
- HP에 따른 회피/후퇴

2. 탐색 AI
- 지정 방향으로 자동 이동
- 위험 감지 시 정지
- 함정/비밀문 자동 탐지
- 시야 내 아이템 자동 수집
- 안전 지역 자동 탐색

3. 생존 관리
- HP 임계치에 따른 회복
- 피폭 수치에 따른 대피
- 기본 자원 자동 활용
- 위험 상황 자동 회피
