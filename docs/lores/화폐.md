# 화폐

## 영혼석의 용도와 거래 시스템

## 1. 영혼석의 실용적 용도

### 마나 저항
- 몸에 지니고 다니면 마나 저항력 부여
- 기본 영혼석 = 마나 저항 10% 상승
- 중급 영혼석 = 마나 저항 30% 상승
- 고급 영혼석 = 마나 저항 50% 상승
- 순수 영혼석 = 마나 저항 80% 상승

### 부활 보조
- 사망 시 부활 성공률 증가
- 기본 영혼석 = 부활 성공률 20% 증가
- 중급 영혼석 = 부활 성공률 40% 증가
- 고급 영혼석 = 부활 성공률 70% 증가
- 순수 영혼석 = 부활 성공률 100% 보장

### 정화 도구
- 오염된 물/음식 정화에 사용
- 기본 영혼석 = 1리터 정화 가능
- 중급 영혼석 = 100리터 정화 가능
- 고급 영혼석 = 1000리터 정화 가능
- 순수 영혼석 = 무제한 정화 가능 (소모되지 않음)

### 마법 증폭
- 주술사의 마법 의식 증폭
- 기본 영혼석 = 마법 위력 20% 증가
- 중급 영혼석 = 마법 위력 50% 증가
- 고급 영혼석 = 마법 위력 100% 증가
- 순수 영혼석 = 마법 위력 300% 증가

## 2. 영혼석의 종류와 가치

### 기본 영혼석 (Basic Soul Stone)
- 크기: 쌀알 크기
- 순도: 30-50%
- 빛: 희미한 청백색
- 가치: 기본 화폐 단위 (1 BSS)
- 지속시간: 1년 (이후 흐려짐)
- 사용 횟수: 최대 10회

### 중급 영혼석 (Medium Soul Stone)
- 크기: 콩알 크기
- 순도: 51-70%
- 빛: 밝은 청백색
- 가치: 100 BSS
- 지속시간: 3년
- 사용 횟수: 최대 50회

### 고급 영혼석 (Premium Soul Stone)
- 크기: 엄지손톱 크기
- 순도: 71-90%
- 빛: 강한 청백색 
- 가치: 1,000 BSS
- 지속시간: 10년
- 사용 횟수: 최대 200회

### 순수 영혼석 (Pure Soul Stone)
- 크기: 손가락 마디 크기
- 순도: 91% 이상
- 빛: 강렬한 백색
- 가치: 10,000 BSS
- 지속시간: 반영구적
- 사용 횟수: 무제한

## 3. 실제 활용 사례

### 일상적 사용
- 식수 정화 (1 BSS로 하루 식수 정화)
- 마나 저항 부적 제작 (10 BSS)
- 간단한 치유 의식 (5 BSS)
- 변이 감지 (3 BSS)

### 전문적 활용
- 대규모 정화 의식 (2 MSS)
- 변이체 사냥 무기 강화 (3 MSS)
- 부활 의식 보조 (5 MSS)
- 마나 폭풍 방어막 생성 (10 MSS)

### 특수 용도
- 영구 결계 설치 (2 PSS)
- 대규모 정화 시설 가동 (3 PSS)
- 변이체 정화 의식 (4 PSS)
- 영혼 재생 의식 (5 PSS)

## 4. 소모와 재생

### 소모 규칙
- 사용할 때마다 빛이 희미해짐
- 정해진 사용 횟수 초과 시 파괴
- 과도한 마나에 노출 시 순도 감소
- 부적절한 보관 시 수명 감소

### 재생 방법
- 달빛 아래 보관 시 매월 1회 충전
- 정화 의식으로 순도 10% 회복 가능
- 다른 영혼석 희생으로 수명 연장
- 순수 영혼 보유자의 축복으로 완전 재생

## 5. 거래 시 주의사항

### 품질 확인
- 빛의 강도 측정
- 잔여 사용 횟수 확인
- 순도 검증
- 재생 이력 확인

### 보관 방법
- 마나 차단 주머니 사용
- 달빛 노출 시간 관리
- 다른 영혼석과 격리
- 온습도 관리
