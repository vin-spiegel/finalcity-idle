# 파이널시티 (Final City) — PRD v1.0

## 1. 개요

브라우저 기반 방치형(Idle) MMO. 포스트아포칼립스 세계에서 구역을 탐색하고, 자원을 수집하며, AI NPC와 상호작용하는 텍스트 + 픽셀아트 하이브리드 게임.

**핵심 차별점:** 장소 기반 탐색이 코어 루프이며, AI NPC가 서사와 이벤트를 생성하여 반복 탐색에 변동성을 부여.

**레퍼런스:** Cybercode Online (구조), IdleMMO (UI), Caves of Qud (감성), 일랜시아 (직업/잡포 시스템)

**기술 스택:** Vite + React, Hono, PostgreSQL, Railway

---

## 2. 코어 루프

```
구획 선택 → 자동 탐색 (틱 기반) → 자원 획득 + 잡포 상승 → 제작/강화/거래 → 더 어려운 구획 탐색
```

### 2.1 탐색 흐름

1. 유저가 구획(Sector)을 선택
2. 서버에서 자동 탐색 시작 (tick_interval 기반)
3. 틱마다: 자원 획득(확정) + 이벤트 발생(확률)
4. 구획 탐색률 100% 도달 시 파밍 모드로 자동 전환 (중단 없음)
5. 유저 복귀 시: `now - last_synced_at` 경과 시간 기반으로 서버에서 일괄 계산

### 2.2 의사결정 포인트 (유저가 접속할 때 하는 일)

- 다음 구획 선택 (또는 현재 구획 반복 파밍)
- 직업 전환 여부 판단
- 미확인 아이템/이벤트 확인
- NPC 거래/의뢰 응답
- 자원 소비 (제작, 강화, 영지 업그레이드)

## 3. 세계 구조

### 3.1 계층

```
월드맵
└── 지역 (Region): 키르타스 평원, ...
    └── 구역 (Zone): 야영지 3구역, ...
        └── 구획 (Sector): 상업 구획 폐건물, 마나 균열 지대, ...
```

### 3.2 구획 (Sector) 속성

| 속성 | 설명 |
|------|------|
| name | 구획 이름 |
| level_req | 진입 최소 레벨 |
| tick_interval | 틱 간격 (초) |
| danger_level | 안전/보통/위험/극한 |
| job_type | 이 구획에서 상승하는 잡포 종류 |
| drop_table | 드롭 테이블 참조 |
| event_table | 이벤트 테이블 참조 |

### 3.3 구획 예시 — 야영지 3구역

| 구획 | Lv | 틱 | 위험도 | 잡포 | 주요 드롭 |
|------|----|----|--------|------|----------|
| 상업 구획 폐건물 | 1 | 8s | 안전 | 수색자 | 마나결정(기본), BSS |
| 구 제조 공장 지하 | 5 | 18s | 보통 | 기술자 | 고철 부품, 설계도 파편 |
| 마나 균열 지대 | 12 | 25s | 위험 | 학자 | 마나결정(중급), 유물 파편 |
| 고대 연구소 잔해 | 20 | 35s | 위험 | 학자 | 마나결정(고급), 고대 기록 |
| 공허 구역 심층부 | 30 | 50s | 극한 | 수색자 | 희귀 유물, 변이 소재 |

## 4. 직업 & 잡포인트 시스템

### 4.1 개요

- 직업은 탐색 스타일을 결정하며, 구획 탐색 시 해당 직업의 잡포인트가 상승
- 잡포인트가 올라갈 때마다 스탯이 자동 증감 (트레이드오프)
- 직업 전환 가능하나 비용 존재
- 직업 순서(루트)에 따라 최종 스탯이 달라짐 → 빌드 전략

### 4.2 1차 직업

#### 수색자 (Searcher)
- 전문: 폐건물 수색, 은닉물 탐지
- 잡포 3마다 탐지력 +1 (최대 30)
- 잡포 10마다 체력 +1 (최대 10)
- 잡포 5마다 마나감응 -1 (최대 -10)
- 해금 스킬: 잡포 20 → 은닉물 감지, 잡포 75 → 위험 예측

#### 학자 (Scholar)
- 전문: 마나 분석, 유물 연구
- 잡포 3마다 마나감응 +1 (최대 30)
- 잡포 5마다 탐지력 +1 (최대 10)
- 잡포 5마다 체력 -1 (최대 -10)
- 해금 스킬: 잡포 20 → 유물 감정, 잡포 75 → 마나 정제

#### 기술자 (Technician)
- 전문: 기계 해체, 장비 제작
- 잡포 3마다 제작력 +1 (최대 30)
- 잡포 10마다 체력 +1 (최대 10)
- 잡포 5마다 마나감응 -1 (최대 -10)
- 해금 스킬: 잡포 20 → 기계 해체, 잡포 75 → 장비 강화

#### 교역가 (Trader)
- 전문: NPC 교류, 자원 교환
- 잡포 4마다 올스탯 +1 (최대 15 각)
- 해금 스킬: 잡포 20 → 흥정, 잡포 75 → 밀수로 개방

### 4.3 직업 전환 규칙

- 전환 시 이전 직업 잡포인트 50% 감소
- BSS 비용 발생
- 쿨다운 24시간
- 잡포인트 100 달성 시 마스터 → 해당 직업 스탯 보너스 영구 유지

### 4.4 2차 직업 (추후)

- 1차 직업 2개 이상 마스터 시 해금
- 예: 수색자 마스터 + 학자 마스터 → 마나 공학자
- 구체적 설계는 1차 직업 밸런스 확인 후

## 5. 자원 & 경제

### 5.1 자원 종류

| 자원 | 획득처 | 용도 |
|------|--------|------|
| BSS (영혼석) | 모든 구획 | 범용 화폐, 직업 전환, NPC 거래 |
| 마나 결정 (기본/중급/고급) | 마나 관련 구획 | 장비 강화, 마나 정제, 구역 해금 |
| 고철 부품 | 공장/폐건물 구획 | 장비 제작 재료 |
| 유물 파편 | 연구소/위험 구획 | 유물 복원, 고급 제작 |
| 변이 소재 | 극한 구획 | 희귀 장비 제작 |
| 설계도 파편 | 공장 구획 | 레시피 해금 |

### 5.2 싱크 (자원 소비처)

- **장비 제작:** 고철 부품 + 설계도 → 장비 (탐색 효율 증가)
- **장비 강화:** 마나 결정 소모 (실패 시 소멸 리스크)
- **직업 전환:** BSS 소모
- **NPC 투자:** 방랑상인에게 자본 투자 → 더 좋은 물건 입고
- **영지 업그레이드:** 자원 소모 → 패시브 보너스
- **구역 해금:** 마나 결정 + 유물 파편 소모

## 6. 틱 시스템

### 6.1 틱 처리 (서버)

```
틱 발생 시:
1. 자원 획득 (드롭 테이블 기반, 확정)
2. 잡포인트 +1 (해당 구획의 job_type)
3. 잡포 임계값 도달 시 스탯 자동 증감
4. 탐색률 증가
5. 이벤트 판정 (확률적)
   - 일반: 없음
   - 특수: 미확인 아이템, NPC 도착, 위험 조우 등
6. 로그 기록
```

### 6.2 오프라인 계산 (복귀 시)

```
경과 시간 = now - last_synced_at
총 틱 수 = floor(경과 시간 / tick_interval)

for each tick:
  자원 += 드롭량
  잡포 += 1
  스탯 재계산
  탐색률 += 틱당 증가분
  이벤트 판정 (랜덤 시드 기반)

last_synced_at = now
```

### 6.3 포어그라운드 동기화

- `visibilitychange` 이벤트로 포어그라운드 복귀 감지
- 복귀 시 서버에 `POST /api/sync` 요청
- 포어그라운드 중에도 30초 간격 폴링
- 클라이언트는 로컬 틱 시뮬레이션 (시각적 피드백용), 서버 응답으로 보정

## 7. 이벤트 시스템

### 7.1 이벤트 유형

| 유형 | 트리거 | UI 처리 |
|------|--------|---------|
| 자원 발견 | 틱 확률 | 로그에 표시 |
| 미확인 아이템 | 틱 확률 (희귀) | 인벤토리에 미확인 상태로 추가 |
| NPC 도착 | 틱 확률 | 채팅에 대화 + 얼럿 배너 |
| 변이체 조우 | 위험 구획에서 틱 확률 | 로그 + 자동 판정 (스탯 기반) |
| 마나 폭풍 | 시간 기반 글로벌 이벤트 | 얼럿 배너 (전체 유저) |
| 잠긴 금고 | 틱 확률 (특정 구획) | 복귀 시 해제 선택지 |

### 7.2 NPC 이벤트 처리

```
NPC 도착 이벤트:
1. 서버가 NPC 유형 + 상품/의뢰 데이터 생성 (서버 로직)
2. 채팅에 AI 생성 인사말 표시 (AI)
3. 유저 액션 시:
   - 거래: 중앙 오버레이 UI (서버 로직으로 가격/재고 처리)
   - 흥정 대사: AI 생성
   - 의뢰 수락: 서버에 퀘스트 생성
```

## 8. NPC 시스템

### 8.1 NPC 유형

| NPC | 역할 | 채팅 | 게임 액션 |
|-----|------|------|----------|
| 방랑상인 | 아이템 거래 | AI 대사 | 거래 UI 오버레이 |
| 순찰대원 | 구역 정보 제공 | AI 대사 | 구역 정보 패널 업데이트 |
| 야영지 주민 | 세계관 서사 | AI 대사 | 없음 (대화 콘텐츠) |
| 수집가 | 특정 아이템 매입 | AI 대사 | 납품 UI |
| 기공사 | 장비 수리/강화 | AI 대사 | 수리/강화 UI |
| 변이체 정찰병 | 위협 경고 | 채팅 경고 | 얼럿 배너 + 방어 선택지 |

### 8.2 AI vs 서버 역할 분리

- **서버 로직:** NPC 등장 타이밍, 상품 목록, 가격 계산, 거래 처리
- **AI 생성:** 대사, 흥정 반응, 이전 대화 기억, 캐릭터 성격
- NPC가 이전 거래를 기억하여 관계 변화 (AI 컨텍스트)

## 9. UI 구조

### 9.1 레이아웃 (데스크톱)

```
[헤더: 캐릭터 :: 스탯 :: 자원 :: 현재 탐색 상태(틱 애니) :: 🔔 알림]
[프로그레스 바 — 전체 폭, 얇게, 텍스트 없음]
[얼럿 배너 — 이벤트 시에만 표시]
[중앙: 픽셀아트 맵 + HUD 오버레이     |  채팅/로그 사이드패널 (접기 가능)]
[  상단 - 구획명, 탐색률, 타이머       |  [🔔 알림 | 📋 로그 | 💬 채팅] ]
[  중앙 - 프로시저럴 픽셀아트 맵       |                                ]
[  하단 - 최근 로그 3~5줄 오버레이     |                                ]
```

### 9.2 하단 탭 바

```
[🗺 지도]  [📦 인벤토리]  [👤 프로필]
```

- 지도: 메인 화면 (구역/구획 선택 + 탐색)
- 인벤토리: 아이템 + 장비
- 프로필: 캐릭터 스탯 + 직업/잡포 + 스킬

### 9.3 네비게이션

- 장소 기반 브레드크럼: `← 키르타스 평원 › 야영지 3구역 › 상업 구획`
- 좌측 사이드바 없음 — 모든 네비게이션은 탭 + 브레드크럼
- 리사이저블 패널 (react-resizable-panels) 으로 중앙/채팅 너비 조절

### 9.4 모바일

- 동일한 구조, 리사이저블 패널로 대응
- 채팅: 플로팅 버튼 → 바텀시트
- 하단 탭 바 유지

### 9.5 아트 디렉션

- 다크 테마 + 모노스페이스 + 사이안/네온 하이라이트
- CoQ 감성: `===` 구분선, `>` 마커, `::` 구분자
- 구획 썸네일: ASCII 아트 (작게)
- 지도: 프로시저럴 픽셀아트 (시드 기반 생성)
- 탐색률에 따라 맵 안개(fog of war) 해제 → 시각적 보상
- UI는 하드코어, 위트는 텍스트(아이템 설명, NPC 대사, 로그)에서
