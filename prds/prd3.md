## 1. 세계관 요약

**배경**: 마나 대폭발 이후 붕괴된 메가시티 "파이널시티". 살아남은 자들은 폐허 속에서 자원을 수집하고, 마나를 연구하며, 도시 재건을 꿈꾼다.

**플레이어 역할**: 순환회(Circulation Guild)에 소속된 탐색자. 구역을 돌며 자원을 수집하고, 스킬을 쌓고, 폐허의 비밀을 해명한다.

**핵심 루프**:
```
구역 탐색 → 자원 수집 → 스킬 성장 → 새 구역 해금 → 더 희귀한 자원
```

## 2. BSS 통화 시스템

### 2.1 BSS란?

**BSS (Block Signal Shard)** — 마나 폭발로 파괴된 도시 통신망의 파편. 희소하며 모든 거래의 기준 통화.

- 드롭: 고위험 구역에서만, 낮은 확률
- 용도: 장비 강화, 직업 전환, 특수 거래, 향후 길드 활동
- 목표 누적 속도: 일 1~5 BSS (일반 플레이어 기준)

### 2.2 BSS 획득처

| 구역 | 드롭률 | 기대값(1회 탐색) |
|------|--------|-----------------|
| 마나 균열 지대 | 3% | 0.03 BSS |
| 고대 연구소 잔해 | 8% | 0.08 BSS |
| 공허 구역 심층부 | 15% | 0.15 BSS |
| *(예정)* 변이체 서식지 | 25% | 0.25 BSS |

> BSS는 탐색 완료(progress 100%)시 드롭. 틱마다 드롭하지 않음.

## 3. 구역 설계 (전체 맵)

### 3.1 구역 계층 구조

```
세계 지도
├── 도심 폐허 지대         [branch]
│   ├── 상업 구획 폐건물   [leaf] 탐험  8s  안전
│   ├── 구 제조 공장 지하  [leaf] 작업  18s 보통
│   └── 공허 구역 심층부   [leaf] 탐험  50s 극한  (searcher 20 필요)
│
├── 마나 연구 구역         [branch]
│   ├── 마나 균열 지대     [leaf] 조사  25s 위험
│   └── 고대 연구소 잔해   [leaf] 조사  35s 위험  (scholar 15 필요)
│
├── 외곽 자연 구역         [branch] ← Phase B
│   ├── 고사목 숲          [leaf] 벌목  20s 안전  (lumberjack 0)
│   ├── 변이 식물군락      [leaf] 벌목  40s 보통  (lumberjack 10 필요)
│   └── 금단의 숲 심층부   [leaf] 벌목  60s 위험  (lumberjack 30 필요)
│
└── 지하 광맥 구역         [branch] ← Phase B
    ├── 얕은 채석장        [leaf] 채굴  25s 보통  (miner 0)
    ├── 마나석 광맥        [leaf] 채굴  45s 위험  (miner 15 필요)
    └── 심층 결정 동굴     [leaf] 채굴  70s 극한  (miner 35 필요)
```

## 4. 자원 & 드롭 테이블

### 4.1 자원 등급 체계

| 등급 | 색상 변수 | 예시 |
|------|----------|------|
| common | `--text-dim` | 마나 결정(기본), 고철 부품 |
| uncommon | `--color-highlight` | 마나 결정(중급), 설계도 파편 |
| rare | `--cyan-dim` | 마나 결정(고급), 고대 기록, 희귀 유물 |
| epic | `--amber` | 변이체 재료, 결정 코어 |
| BSS | 전용 색상 `#ffd700` | BSS |

### 4.2 구역별 드롭 테이블 (1틱 기준, 확률)

**상업 구획 폐건물** (8s)
```
마나 결정(기본)  70%  qty: 1~3
고철 부품        40%  qty: 1~2
설계도 파편      5%   qty: 1
```

**구 제조 공장 지하** (18s)
```
고철 부품        60%  qty: 2~4
설계도 파편      35%  qty: 1~2
마나 결정(중급)  10%  qty: 1
```

**마나 균열 지대** (25s)
```
마나 결정(중급)  55%  qty: 1~2
유물 파편        25%  qty: 1
마나 결정(고급)  8%   qty: 1
BSS              3%   qty: 1  ← 탐색 완료시만
```

**고대 연구소 잔해** (35s)
```
마나 결정(고급)  45%  qty: 1~2
고대 기록        20%  qty: 1
희귀 유물        10%  qty: 1
BSS              8%   qty: 1  ← 탐색 완료시만
```

**공허 구역 심층부** (50s)
```
희귀 유물        35%  qty: 1
변이체 재료      25%  qty: 1~2
마나 결정(고급)  15%  qty: 2~3
BSS              15%  qty: 1  ← 탐색 완료시만
```

## 5. 스킬 시스템 확장 (prd2 기반)

### 5.1 Phase B 신규 스킬

| jobType | actionType | 설명 |
|---------|------------|------|
| lumberjack | 벌목 | 목재·식물 자원 채취 |
| miner | 채굴 | 광물·마나석 채굴 |

**lumberjack (벌목꾼)**:
- +3.00마다: 채취력 +1 (최대 +30)
- +5.00마다: 체력 +1 (최대 +10)
- 10.00 달성: 해금 — 변이 식물군락
- 30.00 달성: 해금 — 금단의 숲 심층부

**miner (채굴꾼)**:
- +3.00마다: 채굴력 +1 (최대 +30)
- +10.00마다: 체력 +1 (최대 +10)
- 15.00 달성: 해금 — 마나석 광맥
- 35.00 달성: 해금 — 심층 결정 동굴

### 5.2 스킬 해금 로그

특정 마일스톤 달성 시 로그에 표시:
```
[탐험] 20.00 달성 — 은닉물 감지 해금
[조사] 15.00 달성 — 고대 연구소 진입 가능
```

구현: `jobPointsGained`로 마일스톤 크로스 감지 후 `ADD_LOG` 디스패치.

## 6. 인벤토리 & 아이템 시스템

### 6.1 현재 상태

- 인벤토리: 클라이언트 하드코딩 (`GameContext.tsx`)
- 실제 드롭은 `resources` (단순 카운터) 로만 존재
- 둘의 연동 없음

### 6.2 목표 구조

```
resources (DB)         ← 탐색으로 누적되는 숫자
inventory (UI)         ← resources를 아이템 형태로 표시
```

**resources 테이블** → 각 자원 key별 qty
**inventory UI** → resources를 RESOURCE_NAMES 매핑으로 아이템 카드 렌더

구현 순서:
1. `resources` API 반환을 init에 포함 (이미 구현)
2. 인벤토리 탭에서 resources 기반으로 렌더
3. 하드코딩된 `inventory` 배열 제거

### 6.3 인벤토리 탭 설계

```
┌─────────────────────────────────────────────────┐
│ 인벤토리                          필터: [전체 ▾] │
├──────────────────────────────────────────────────┤
│ 마나 결정(기본)  common   ×124               ... │
│ 고철 부품        common   ×45                ... │
│ 설계도 파편      uncommon ×8                 ... │
│ 마나 결정(중급)  uncommon ×14               ... │
│ BSS              special  ×3                 ... │
└──────────────────────────────────────────────────┘
```

## 7. UI 설계 보완

### 7.1 스킬 탭 (Phase A)

ProfileView 내 또는 별도 탭:

```
스킬                   숙련도       진행도
─────────────────────────────────────────────────
탐험 (수색자)          3.45    ████░░░░░░  3.45/20.00 (다음: 은닉물 감지)
작업 (기술자)          0.00    ░░░░░░░░░░  미숙련
조사 (학자)            0.00    ░░░░░░░░░░  미숙련
```

- 마일스톤까지 남은 양 표시
- 0.00인 스킬은 흐리게 처리 (`opacity: 0.45`)

### 7.2 구역 리스트 개선

branch 뷰에서 잠금 표시 개선:

```
상업 구획 폐건물    탐험  ◷ 8s   ■ 안전        ▸
공허 구역 심층부    탐험  ◷ 50s  ■ 극한   🔒   ▸
```

- 잠금 구역: 클릭하면 리프 뷰로 진입, 잠금 이유 표시
- 잠금 해제 조건 hover시 툴팁 (모바일 불가 → tap)

### 7.3 탐색 완료 이벤트

progress 100% 달성 시:
- 로그에 `[탐험 완료] 공허 구역 심층부 탐색 완료` 표시
- BSS 드롭 있으면 `🏆 BSS ×1 획득` 강조 로그
